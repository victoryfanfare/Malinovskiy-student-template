# HW05 - Линейные модели и честный ML-эксперимент

## Описание
Домашнее задание по семинару S05. Цель - закрепить навыки работы с линейными моделями (логистическая регрессия), бейзлайнами и проведением честного ML-эксперимента.

## Структура
- `HW05.ipynb` - основной ноутбук с решением
- `figures/` - сохраненные графики
- `artifacts/` - сохраненные модели и метрики

## Результаты эксперимента
1. Загружен и проанализирован датасет для предсказания дефолта по кредиту
2. Построен бейзлайн (DummyClassifier) с accuracy ~0.51
3. Обучена логистическая регрессия с подбором гиперпараметров
4. Достигнуто улучшение ROC-AUC с 0.42 до 0.87
5. Сохранены все артефакты эксперимента
